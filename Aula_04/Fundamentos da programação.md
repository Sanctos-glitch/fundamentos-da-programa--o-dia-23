linguagem de programação tipada é o momento que vc esta declarando uma variável e você guarda um dado, no momento que você esta declarando você precisa falar o tipo de variável que você vai receber, isso são programações tipadas.



basicamente tipar é dizer o tipo.



quando você vai declarar uma variável você precisa usar um padrão, existem 2 tipos mas hoje vamos usar o padrão **snake\_case.**



no padrão snake\_case qualquer tipo de espaço entre palavras precisam ser separadas com "\_" e não com um espaço em branco.



toda vez que você for nomear alo você não usa "do,de,da,di,"; você só pula isso pra evitar sobrecarregamento indesejado exemplo: 



##### ao invés de : **nome\_do\_usuario**

Fstring

&nbsp;quando você coloca uma frase usando as " dentro do parênteses do print, ao tentar fazer a operação matemática, o vscode pode considerar aquilo como um comentário e não uma operação. Para mudar isso usamos o fstring dessa forma:



digamos que a frase seja "A soma entre numero\_um = numero\_dois = soma"



para usar fstring, coloque antes das " um ''f''



agora o fstrinf esta ativado, e tudo oq você precisa fazer é colocar um {} nas frases que voce quer que aconteça a operação, ficando assim:



print(f"A soma entre {numero\_um} + {numero\_dois} = {soma}")
